```
WHAT TO DO:
  Extract: course names, fees, dates, key facts, USPs, contact info,
```

```
  Use SVG for: icons, charts, data visualisations, decorative elements

  Maximum 40 words of body text per slide
  Statistics shown large — 48px minimum for key numbers
  Every third slide must have a strong visual moment — full bleed or infographic
  Final slide always: clear CTA with contact information
```

## DESIGN SYSTEM — DEASSISTS BRAND LOCKED

### Typography — non-negotiable

```
DISPLAY / HEADLINES:
  Font: Fraunces (serif)
  Weight: 700
  Use: slide titles, hero statements, key statistics
  Colour: #0d1a10 (dark) or #ffffff (on dark backgrounds)
  Size: 36–56px for titles, 28–36px for section headers

BODY / SUPPORTING:
  Font: Outfit (sans-serif)
  Weight: 400 regular, 600 semi-bold for labels
  Use: body copy, captions, labels, bullets, metadata
  Colour: #1a1a1a primary, #888888 muted
  Size: 14–16px body, 12px captions, 11px labels

STATISTICS (the hero number):
  Font: Fraunces
  Weight: 700
  Size: 48–72px — statistics must command attention
  Colour: #1D7A45 green always for positive stats
```

### Colour System — brand tokens

```
PRIMARY GREEN:    #1D7A45  — primary actions, headlines, positive data
GREEN HOVER:      #2a9458  — interactive states
GREEN LIGHT BG:   #e8f5ee  — section backgrounds, highlights
PAGE BG:          #F6F7F4  — warm cream — never pure white
CARD BG:          #ffffff  — content areas
TEXT PRIMARY:     #1a1a1a  — main body text
TEXT MUTED:       #888888  — captions, metadata
BORDER:           #e5e5e0  — dividers, card borders
AMBER ACCENT:     #c47b00  — attention, call-outs, warnings
DARK HERO:        #0d1a10  — dark slide backgrounds, hero sections
```

### Semantic Colour Language — locked

```
GREEN  #1D7A45 = positive, active, growth, achievement, CTA
AMBER  #c47b00 = call-out, highlight, important note
GREY   #888888 = supporting, metadata, secondary information
RED    #c62828 = use sparingly — problems only, never decorative
WHITE  #ffffff = content, breathing room, card backgrounds
DARK   #0d1a10 = authority, hero sections, premium feel
```

### Slide Layout Principles

```
HERO SLIDES (first slide, section dividers):
  Full bleed dark background #0d1a10
  Fraunces white headline — large
  Green #1D7A45 accent element
  Minimal text — maximum impact

CONTENT SLIDES (information delivery):
  Cream #F6F7F4 background
  Clear section label — Outfit 11px uppercase
  One primary message per slide
  Data visualised — not listed

DATA SLIDES (statistics, comparisons):
  Large Fraunces number in green
  Context below in Outfit 14px
  Chart or infographic where possible
  Source cited in Outfit 11px muted

CLOSING SLIDES (CTA, contact):
  Green #1D7A45 background
  White Fraunces headline
  Contact details clear and prominent
  One clear next action stated

SPACING RULES:
  Margin: 60px all sides minimum
  Element spacing: 24px between sections
  Never crowd — white space is intentional design
```
