#### 2.2 Modified files (Stage 2 — harness carve-out, in-place edit)

**`harness/core/patcher.js`** — replace the hardcoded `NEVER_TOUCH_SUBSTRINGS` const (lines 18–35) with a parser that reads `369-brain/CODING-CONSTITUTION.md` § A7 once at module load.

Parser algorithm:
1. Read constitution file from `path.join(__dirname, '..', '..', 'CODING-CONSTITUTION.md')`.
2. Locate heading `### A7 — Never-Touch Files`.
3. Extract section body up to the next `### ` heading.
4. Iterate each line. Skip if:
   - Empty / whitespace-only
   - Starts with `#`
   - Starts with `These files cannot`, `Under any`, `If you find`, `Bring it to`
   - Starts with `Any ` (semantic line, not a substring — e.g., "Any Stripe or payment logic")
   - Is a section label ending with `:` (e.g., `PRODUCTION REPO (deassists):`)
5. For surviving lines:
   - Trim leading whitespace
   - Strip trailing parenthetical comments: `package.json and pnpm-lock.yaml (without Latha approval)` → `package.json and pnpm-lock.yaml`; `.env files (read only)` → `.env files`
   - Strip trailing ` files` if present (matches `.env files` → `.env`)
   - Split on ` and ` to handle compound entries
   - Validate each token matches `/^[\w\-./]+$/` — if not, drop it
6. Result is exposed via `getNeverTouchSubstrings()` (cached after first call).
7. **Fail-loud:** if the file cannot be read, the heading cannot be found, or the parsed list is empty, throw `Error('NEVER-TOUCH parse failed: <detail>')` at first call. Module load itself does not parse — parsing happens on first invocation of `validatePatch` so the harness can still load for non-validating tasks. (This deliberate deviation from "throw at module load" in Mode 1 keeps test isolation: tests can mock the constitution file before first call.)
8. The parsed list MUST be a superset of these 11 entries (Stage 1 test asserts this exactly):
   ```
   apps/cms-next/pages/universitiesd/
   apps/backend-nest/src/core/entities/extendables/payment.entity.ts
   apps/mui-cms-next/
   MASTER-RUN.cjs
   scope.guard.ts
   package.json
   pnpm-lock.yaml
   .env
   archive/
   code-snapshot/
   graphify-out/
   ```
9. Public surface: `validatePatch()` calls `getNeverTouchSubstrings()` instead of reading a const. The exported name `NEVER_TOUCH_SUBSTRINGS` is preserved for backward-compatibility but becomes a getter (Object.defineProperty on `module.exports`) that calls `getNeverTouchSubstrings()` lazily. Existing imports continue to work without reshape.
